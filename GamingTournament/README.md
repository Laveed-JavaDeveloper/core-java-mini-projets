# Gaming Tournament

A Core Java implementation of an algorithm that generates the maximum possible entrance code by combining digits from two input sequences.

## Features

* Combines two input lists
* Generates the largest possible sequence
* Efficient greedy algorithm implementation

## Concepts Used

* Arrays
* ArrayList
* Greedy Algorithm
* Method Decomposition
* Problem Solving

## How to Run

1. Open the project in your Java IDE.
2. Execute the main program or integrate the class into your own application.
3. Provide input lists and the desired output length.

## Learning Outcome

This project focuses on algorithm design and efficient array manipulation using Core Java.
