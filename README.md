# Core Java Mini Projects

A collection of console-based Java applications developed to strengthen Core Java fundamentals, Object-Oriented Programming (OOP), Java Collections Framework, and problem-solving skills.

Each project focuses on applying Java concepts through practical implementations.

---

## Projects

| Project | Description | Concepts |
|---------|-------------|----------|
| **Calculator App** | Performs basic arithmetic operations with input validation. | Methods, Operators, Conditional Statements |
| **Guesser Game** | Number guessing game involving multiple players and an umpire. | Classes & Objects, Encapsulation, Decision Making |
| **To-Do List App** | Console-based task manager supporting CRUD operations. | ArrayList, Collections Framework, User Input |
| **Gaming Tournament** | Algorithm-based solution for generating the maximum entrance code. | Arrays, Collections, Greedy Algorithm |
| **Smart Elevator System** | Simulation of a multi-elevator system using OOP and the Strategy Design Pattern. | OOP, Strategy Pattern, Polymorphism, Abstraction |

---
