# To-Do List App

A console-based task management application that allows users to manage daily tasks through a simple menu-driven interface.

## Features

* View Tasks
* Add Task
* Edit Task
* Delete Task
* Menu-driven navigation

## Concepts Used

* ArrayList
* CRUD Operations
* Loops
* Conditional Statements
* User Input (`Scanner`)

## How to Run

1. Open the project in your Java IDE.
2. Run `ToDoListApp.java`.
3. Choose an option from the menu.

## Sample Menu

```
1. View Tasks
2. Add Task
3. Edit Task
4. Delete Task
5. Exit
```
