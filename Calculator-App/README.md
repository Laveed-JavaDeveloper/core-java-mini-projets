# Calculator App

A simple console-based calculator built using Core Java. It performs basic arithmetic operations and validates user input.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Modulus
* Invalid operator handling

## Concepts Used

* Methods
* Conditional Statements
* User Input (`Scanner`)
* Basic Validation

## How to Run

1. Open the project in your Java IDE.
2. Run `Calculator.java`.
3. Enter two numbers and an operator when prompted.

## Sample Output

```
========= Simple Calculator =========

Enter Number 1: 15
Enter Operator (+, -, *, /, %): *
Enter Number 2: 8

Result: 120.0
```
