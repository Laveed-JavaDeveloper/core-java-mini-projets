# Smart Elevator System

A console-based simulation of a smart elevator system developed using Object-Oriented Programming principles and the Strategy Design Pattern.

## Features

* Multiple elevators
* Nearest elevator selection
* Floor movement simulation
* Elevator state management
* Console-based animation

## Concepts Used

* Object-Oriented Programming (OOP)
* Encapsulation
* Abstraction
* Polymorphism
* Interfaces
* Enums
* Strategy Design Pattern

## Project Structure

```
Smart-Elevator-System
│
├── Elevator.java
├── ElevatorController.java
├── DispatchStrategy.java
├── ClosestElevatorStrategy.java
├── State.java
└── SmartElevatorSimulation.java
```

## How to Run

1. Open the project in your Java IDE.
2. Run `SmartElevatorSimulation.java`.
3. Enter a floor number between 0 and 10.
