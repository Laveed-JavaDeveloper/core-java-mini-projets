# Guesser Game

A console-based number guessing game where one player acts as the guesser and three players attempt to guess the same number. The umpire determines the winner(s).

## Features

* Guesser selects a number
* Three players submit guesses
* Supports multiple winners
* Displays the final result

## Concepts Used

* Classes and Objects
* Encapsulation
* Method Calling
* Conditional Logic
* Object Interaction

## How to Run

1. Open the project in your Java IDE.
2. Run `GuesserGame.java`.
3. Enter the guesses when prompted.

## Sample Output

```
Guesser guess the number: 7

Player1: 7
Player2: 5
Player3: 7

Winner: Player 1 & Player 3
```
